# Bazy-danych - Projekt
Projekt z laboratoriów z baz danych - baza danych filmowych z interfacem.

Folder *interface* zawiera skrypty pisane w Pythonie dotyczące działania aplikacji
Folder *data* zawiera pliki z danymi filmowymi
Folder *raport* zawiera plik PDF z raportem projektu
Plik *Script.sql* zawiera skrypt pisany w SQL dotyczące bazy danych i jej działania
